GreenLog API Setup

Configure your Groq API key for AI features

ℹ️ What is this?
This tool helps you set up your Groq API key for AI-powered features like photo recognition and smart insights.
📝 How to get your key:
1. Go to console.groq.com
2. Sign up or log in
3. Create an API key
4. Paste it above
⚠️ Security: Your key is stored only in your browser's local storage. Never share it publicly. Clear it if you switch devices.
